Hybrid CME Detection Model

Our detection framework integrates physics-inspired features with advanced AI models to robustly identify CME events from solar wind data.

1. Entropy-Based Preprocessing

  • Shannon & Tsallis Entropy quantify randomness and non-equilibrium behavior in solar wind parameters.
  • Used to detect disorder spikes that often precede CME events.

2. Classical Machine Learning

  • Isolation Forest and Autoencoders trained on normal data detect statistical anomalies.
  • Triggers real-time entropy alerts for further analysis.

3. Deep Learning for Temporal Modeling

  • BiLSTM + Attention captures long-range temporal dependencies in entropy patterns.
  • Attention mechanism helps the model focus on critical time windows where CME signals are prominent.

4. Real-Time Inference & Alerting

  • Live entropy graphs are monitored for spikes beyond threshold (e.g. 1.2).
  • Alerts are triggered based on combined entropy and model anomaly scores.
